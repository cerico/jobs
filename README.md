## How do i run it?


```
npm run dev
```

Starts dev build, runs on http://localhost:3404

```
npm run build
```

Creates production build for express

```
npm run start
```

Starts production build, runs on http://localhost:3333

```
npm run deploy
```

Pushes to DigitalOcean droplet, runs on http://jobs.malham.io
